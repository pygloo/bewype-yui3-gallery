| bewype-yui3-gallery ¶

The YUI3 Bewype Gallery is a fork of the YUI3 framework to manage nice additional widgets, plugins and tools. Then it take advantage of the YUI build mechanism to maintain and deliver these works. Thanks to YUI3 for its "built-in" dynamic modules, translations and theming loading ;)

Here is the list of the main modules available in this gallery project. They use the following namespace Y.Bewype and are the only part of this fork to be maintained. Core YUI3 part must remain clean and correspond with a recent version of the framework.

In the futur, we will try to separate these to parts as a standard gallery. Meanwhile loader issue seems to be the only (or easy) way for very useful dynamic import.

Layout Designer ¶

The Y.Bewype.LayoutDesigner is a tool based on several bewype widgets and plugins. It aims to easily design simple newsletter contents base on table and inline-css.

This feature has been developed for the company Emencia and should be integrate in the Emencia-Django-Newsletter soon. Big thanks to Emencia for its enthousiasm in creating a such nice open-source tool and for its believe in the YUI3 library.
